# LatakhsarRentCars
Latakhsar RentCars adalah sebuah aplikasi rental mobil berbasis dekstop yang saya buat menggunakan bahasa pemrograman java.
This about  my project base on Java Netbean.
# System Requrement
- Git (for cloning project)
- JDK & JRE
- NetBean
- MySQL/SQL
